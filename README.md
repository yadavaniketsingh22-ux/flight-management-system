# ✈️ Flight Management System

## 📌 Project Overview
The Flight Management System is a web-based application developed to manage flight records, bookings, and passenger details efficiently.  
This system helps in organizing flight schedules, managing customer data, and maintaining database records using PHP and MySQL.

---

## 🚀 Features
- Add new flight details  
- Update flight information  
- Delete flight records  
- View available flights  
- Manage passenger data  
- Connected with MySQL database  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- PHP  
- MySQL (XAMPP Server)  

---

## 💻 How to Run the Project
1. Install XAMPP on your system.
2. Move the project folder to the `htdocs` directory inside XAMPP.
3. Start Apache and MySQL from the XAMPP Control Panel.
4. Import the database file into phpMyAdmin.
5. Open your browser and run:
   http://localhost/your-project-folder-name
